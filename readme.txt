SMS уведомления о событиях WooCommerce через шлюз WEBSMS.RU

== Description ==

SMS уведомления о событиях интернет-магазина WooCommerce через шлюз WEBSMS.RU.
[САЙТ](http://websms.ru/services/sending/request) | [Поддержка](support@websms.ru) 

= Features =

* Уведомление о новом заказе на телефон, указанный продавцом.
* Уведомление о новом заказе на телефон покупателя.
* Уведомление о последующих изменениях статуса заказа на телефон, указанный продавцом.
* Уведомление о последующих изменениях статуса заказа на телефон покупателя.

== Installation ==
1. Убедитесь что у вас установлена посленяя версия плагина [WooCommerce](http://www.woothemes.com/woocommerce)
2. Через консоль
  * В административной панели перейдите на страницу Плагины и нажмите Добавить новый.
  * Перейдите на вкладку Загрузить, нажмите Обзор и выберите архив с плагином. Нажмите Установить.
3. По FTP
  * распакуйте архив и загрузите "websms-woocommerce" в папку ваш-домен/wp-content/plugins
4. После того, как плагин будет установлен, нажмите Активировать плагин.
5. Наведите курсор на пункт меню WooCommerce и выберите WEBSMS.RU
6. В настройках введите логин, пароль, телефон продавца.
7. Укажите статусы для каждого вида сообщений, по необходимости, откорректируйте текст SMS-оповещений
8. Сохраните изменения, кликнув на кнопке "Сохранить".
